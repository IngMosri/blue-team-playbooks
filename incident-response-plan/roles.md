# Incident Response Roles

## SOC L1 Analyst
- Monitor security alerts and dashboards
- Perform initial triage and classification
- Escalate confirmed incidents to SOC L2

## SOC L2 Analyst
- Perform deep investigation and root cause analysis
- Contain and mitigate threats
- Coordinate with IT and endpoint teams

## SOC L3 / Security Engineer
- Lead incident response
- Perform forensic analysis
- Improve detection rules and playbooks

## IT Support / Endpoint Team
- Apply remediation actions
- Patch systems and isolate endpoints
- Support user communication

## Management / Legal / HR
- Approve user communication
- Handle compliance and privacy concerns
